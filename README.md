# Office Escape — Level 1

Браузерная 2D-платформер игра (жанр в духе классических сайд-скроллеров), сделанная на чистом HTML/CSS/JS + Canvas, без фреймворков и внешних библиотек.

## Запуск

Просто откройте `index.html` в браузере — либо разместите папку как GitHub Pages (Settings → Pages → Deploy from branch → `/root`).

## Управление

- **Desktop:** `A`/`D` или `←`/`→` — бег, `Space` — прыжок
- **Mobile:** полупрозрачные кнопки ◀ ▶ и кнопка **JUMP** в углах экрана (появляются автоматически на сенсорных устройствах)

## Структура проекта

```
index.html          — разметка экранов (boot / HUD / win / lose) и canvas
css/style.css        — вся визуальная стилизация интерфейса
js/audio.js           — синтезированные звуки (WebAudio), без внешних файлов
js/input.js           — обработка клавиатуры и тач-кнопок
js/entities.js        — классы Player, Enemy, Boss, Collectible + физика
js/level1.js          — данные уровня 1 (платформы, враги, предметы, босс, дверь)
js/renderer.js        — вся отрисовка на canvas
js/game.js            — игровой цикл, камера, коллизии, состояния игры
```

## Как добавить Level 2

1. Создать `js/level2.js` с функцией `instantiateLevel2(groundY)`, которая возвращает объект той же формы, что и `instantiateLevel1` (см. конец `js/level1.js`).
2. Подключить файл в `index.html` рядом с `level1.js`.
3. Зарегистрировать уровень в `LEVELS` в начале `js/game.js`:
   ```js
   const LEVELS = {
     1: instantiateLevel1,
     2: instantiateLevel2
   };
   ```
4. На экране победы включить кнопку `Continue`, чтобы она вызывала `Game.loadLevel(2)` вместо `disabled`.

## Игровые механики

- Стресс (0–20): +1 за столкновение с врагом сбоку, обнуляет прогресс при заполнении («Выгорание»).
- Энергия: восстанавливается кофе (☕ +10, также немного снижает стресс).
- Уверенность/очки: сигареты (🚬) и кофе дают очки, врагов можно «затоптать» прыжком сверху.
- Финальный босс: нужно перепрыгнуть — при касании происходит мгновенное «выгорание».
